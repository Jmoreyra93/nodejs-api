# API de Login de usuarios, creación , eliminación, actualización y obtención completa o filtrada de usuarios.



## Installation

Install the dependencies and devDependencies and start the server.

```sh
cd nodejs-api
npm i
npx nodemon src
```

For production environments...

```sh
npm run dev 
```
